x64 platformlarda ASP.NET 2.0'ı 32-bit moda dönüştürme

64-bit Windows sürümünde WSP kullanılacak ve aynı zamanda 32-bit uygulamalar (örn. posta sunucusu) yönetilecekse, IIS'de ASP.NET 2.0, 32-bit moda dönüştürülmelidir.

 

Öncelikle adsutil.vbs yardımcı programını kullanarak Enable32BitAppOnWin64 ayarını IIS meta tabanında değiştirin. adsutil.vbs komut satırı genellikle c:\Inetpub\AdminScripts klasöründe bulunur, bu nedenle önce şu CD komutu gerçekleştirilmelidir:

Başlat -> Çalıştır... -> cmd -> [ENTER]

C> cd c:\Inetpub\AdminScripts

 

ardından şu komutu çalıştırarak 32-bit IIS ISAPI modülünü etkinleştirin:

cscript.exe adsutil.vbs set W3SVC/AppPools/Enable32BitAppOnWin64 "1"

 

Ardından ASP.NET 2.0 ISAPI kayıtlarını IIS meta tabanında güncelleyin:

C:\WINDOWS\Microsoft.NET\Framework\v2.0.50727\aspnet_regiis.exe -i

 

Internet Bilgi Hizmetleri Yöneticisindeki Web hizmeti uzantı listesinde ASP.NET sürüm 2.0.50727 (32-bit) öğesinin İzin Verilen olarak ayarlandığından emin olun.
