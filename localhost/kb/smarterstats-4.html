SmarterStats 3.x Kurulum ve Yapılandırma

Varsayılan olarak SmarterStats, kurulum sonrasında programı hemen kullanmanızı mümkün kılan temel bir web sunucusu kuracaktır. Ancak SmarterStats uygulamasını, Microsoft Internet Bilgi Sunucusu (IIS) gibi daha sağlam ve güvenli bir web sunucusuna taşımanız tavsiye edilir. SmarterStats programını IIS 5.0 veya daha üzeri ile çalışacak şekilde yapılandırmak için, aşağıdaki adım adım talimatları uygulayın.

 

Bu bölümde SmarterStats programının kendi IIS sitesi olarak kurulması anlatılmaktadır (tavsiye edilen yöntem). SmarterStats programını mevcut bir IIS sitesinde sanal dizin olarak çalıştırmak istiyorsanız burayı tıklatın.

 

Not: Bu başlıkta, IIS ve nasıl çalıştığı konusunda bilgi sahibi olduğunuz varsayılmaktadır. SmarterTools, IIS kullanmak istememeniz halinde SmarterStats ile birlikte verilen web sunucusunu kullanmanız tavsiye edilir.

 

Not: Önceki SmarterStats sürümleri için gerekli olan bazı adımlar artık gerekli değildir ve bu başlıktan çıkarılmıştır.

 

IIS'yi kurun (gerekiyorsa)

Başlat -> Denetim Masası -> Program Ekle/Kaldır öğesini tıklatın

Windows Bileşeni Ekle/Kaldır öğesini tıklatın.

Internet Bilgi Hizmetleri (IIS) kutusunu işaretleyin.

Sihirbazı tamamlayın ve kurulumun tamamlanmasını bekleyin.

Eğer Microsoft.Net eşlemeleri doğru değilse (örneğin Microsoft.Net sonrasında IIS kurmanız halinde), komut isteminde şu komutu çalıştırın (Windows 2000 kurulumlarında "\Windows\" yerine "\WinNT\" ekleyin):

  • c:
  • cd \Windows\Microsoft.Net\Framework\v1.1.4322
  • aspnet_regiis -i

 

SmarterStats Web Sunucusunu Devre Dışı Bırakma

Kesinlikle gerekli olmamakla birlikte bu adım, sunucunun güvenliğini artırır ve tavsiye edilir.

Bu adımlar SmarterStats ile birlikte verilen Web Sunucusunu durduracak ve devre dışı bırakacaktır. Yeni SmarterStats URL'sini güncelleyene kadar masaüstündeki SmarterStats kısayolunun devre dışı kalması şeklinde bir yan etki de söz konusu olacaktır.

Başlat -> Program Dosyaları -> SmarterTools -> SmarterStats -> SmarterStats Web Sunucusunu Yapılandır öğesini tıklatın

Durdur düğmesini tıklatın

Çalıştırma Modunu Devre Dışı olarak değiştirin ve Uygula düğmesini tıklatın

Kapat düğmesini tıklatın

 

SmarterStats uygulamasını IIS'ye ekleyin

Başlat > Denetim Masası > Yönetim Araçları > Internet Bilgi Hizmetleri (IIS) öğesini tıklatın

Sayfanın sol tarafındaki web siteleri ağaç görünümünü sağ tıklatın ve Yeni -> Web Sitesi öğesini seçin

Açıklama istendiğinde SmarterStats girin

Web arabirimi için kullanılacak IP adresini girin. Eğer bu IP adresi başka bir web sitesi ile paylaşılıyorsa, farklı bir port veya Ana Bilgisayar Başlığı kullanmanız gerekecektir. Ana Bilgisayar Başlığı hakkında daha fazla bilgi sahibi olmak için IIS belgelerine bakın.

Yol için "C:\Program Files\SmarterTools\SmarterStats\MRS" kullanın (veya SmarterStats kurduğunuz konumu kullanın).

"Bu web sitesine anonim erişime izin ver" onay kutusunu işaretli olarak bırakın.

Hangi izinlerin verileceği sorulduğunda yalnızca Okuma ve Çalıştırma Komut Dizilerini seçin.

Sihirbazı bitirin

 

IIS Ayarlarını Doğrulama

Eğer Windows 2003 kullanıyorsanız, ASP.NET v1.1.4322 Web Hizmeti Uzantısında IIS için izin Verilmiş olarak ayarlandığını doğrulayın. Eğer ASP.NET öğesi Web Hizmeti Uzantıları altında görülmüyorsa, ASP.NET öğesinin Uygulama Sunucusunda kurulu olduğundan emin olun. (Denetim Masası -> Program Ekle/Kaldır -> Windows Bileşenleri -> Uygulama Sunucusu -> Ayrıntılar)

 

IIS'de SmarterStats sitesi Yapılandırma dizinini sağ tıklatın ve özellikleri seçin. Okuma işaretini kaldırın ve Tamam öğesini tıklatın. Sorulması halinde değişiklikleri alt nesnelere de uygulayın.

Site varsayılan belgesinin Default.aspx olduğundan emin olun (Belgeler sekmesinde)

 

DNS Güncelleme

SmarterStats imleci ile herhangi bir etki alanı ayarlanmışsa, webmail DNS kayıtlarını yeni IP adresini gösterecek şekilde güncelleyin.

 

Siteyi Test Etme

Bir web tarayıcı açın ve eklediğiniz sitenin IP adresini girin.
