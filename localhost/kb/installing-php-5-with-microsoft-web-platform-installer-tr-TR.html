Microsoft Web Platform Installer ile PHP 5 Kurma

Bu makalede IIS 7.x için Microsoft Web Platform Installer ile PHP5 kurulumu anlatılmaktadır

 

1. Microsoft Web Platform Installer uygulamasını http://www.microsoft.com/web/downloads/platform.aspx adresinden yükleyin

2. Başlat > Tüm Programlar > Microsoft Web Platform Installer öğesini açın;

3. Web Platform > PHP 5.x.x öğesini seçin, Kur öğesini tıklatın, I Kabul düğmesini tıklatın, Bitir ve Çık öğesini tıklatın;

PHP için Windows Ön Bellek Uzantıları öğesinin seçilmesi ve kurulması tavsiye edilir.

Windows Ön Bellek Uzantıları hakkında daha fazla bilgiye şuradan ulaşılabilir: http://www.iis.net/download/WinCacheForPhp

4. PHP ayarlarını daha kolay yönetebilmek için, IIS için PHP Yöneticisini http://phpmanager.codeplex.com/releases/ adresinden yükleyebilir ve kurabilirsiniz

5. IIS Yöneticisini açın, ana düğümü seçin, PHP Yöneticisi arayın ve açın.

6. Tavsiyeleri görüntüle bağlantısını tıklatın, tüm seçenekleri seçin ve Tamam öğesini tıklatın;

7. Yapılandırma hatası raporunu seçin, üretim bilgisayarını seçin, uygula öğesini tıklatın;

8. Lütfen şu makaleyi okuyun: http://learn.iis.net/page.aspx/246/using-fastcgi-to-host-php-applications-on-iis-7/ ve gereken diğer özelleştirmeleri yapın.

9. WebsitePanel Sunucusunda WebsitePanel > Web Hizmeti Özellikleri Sayfasına gidin ve PHP 5.x Yürütülebilir Dosya Yolunun doğru ayarlanmış ve FastCGI uygulamasının aşağıdaki resimdeki şekilde seçildiğinden emin olun.

10. NETWORK SERVICE ve WSPWebUsers için PHP kurulum klasöründe Okuma ve Yürütme izinlerinin ayarlanmış olduğundan emin olun (C:\Program Files (x86)\PHP)

IIS için PHP Yöneticisi modülünde diğer PHP sürümlerinin değiştirilmesi veya kaydedilmesi mümkün değildir.
