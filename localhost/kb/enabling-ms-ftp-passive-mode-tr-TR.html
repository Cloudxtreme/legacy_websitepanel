MS FTP Pasif Modunu Etkinleştirme

Doğrudan Meta Tabanı Düzenlemeyi Etkinleştirme:

1.       IIS Microsoft Management Console (MMC) uygulamasını açın.

2.       Yerel Bilgisayar düğümünü sağ tıklatın.

3.       Özellikler öğesini seçin.

4.       Doğrudan Meta Tabanı Düzenlemeyi Etkinleştir onay kutusunun işaretlenmiş olduğundan emin olun.

 

ADSUTIL komut satırı ile PassivePortRange öğesini yapılandırın:

1.       Başlat ve Çalıştır öğesini tıklatın, cmd yazın ve Tamam öğesini tıklatın.

2.       cd Inetpub\AdminScripts yazın ve ENTER tuşuna basın.

3.       Aralığın ".." ile belirlendiği yere şu komutu yazın.
cscript.exe adsutil.vbs set /MSFTPSVC/PassivePortRange "5001-5201"

4.       FTP Yayın Hizmetini Yeniden Başlatın. ADSUTIL komut satırı ile yapılandırma sırasında şu çıktıyı görebilirsiniz:

 

Microsoft (R) Windows Script Host Version 5.6

Copyright (C) Microsoft Corporation 1996-2001. Tüm hakları saklıdır.

 

PassivePortRange : (STRING) "5001-5201"

Komut Satırından Güvenlik Duvarına bir dizi port eklemek için

1.       Başlat ve Çalıştır öğesini tıklatın, cmd yazın ve Tamam öğesini tıklatın.

2.       Aralığın ( ) olarak belirlendiği yere şunu girin, güvenlik duvarı girişi ise " " içindedir.
FOR /L %I IN (5001,1,5201) DO netsh firewall add portopening TCP %I "Passive FTP"%I

3.       Aralıktaki her bir port "Tamam" onayı ile eklenecektir.
