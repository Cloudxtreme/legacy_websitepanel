Could not load file or assembly mlt_ver or one of its dependencies

Translations:

  • tr-TR

Error

After installing MyLittleAdmin by WSP Configuration Studio on Windows Server 2008 X64 with IIS 7 you can receive the following error message:

Could not load file or assembly 'mlt_ver' or one of its dependencies. An attempt was made to load a program with an incorrect format.

Resolution

Open IIS7 Manager, right click "myLittleAdmin Lite Edition Pool" , click Advanced Settings and set Enable 32-Bit Applications to True.
