Creating ODBC Data Source

Translations:

  • tr-TR

Open “Hosting Space home” page.

Some legacy applications require ODBC Data Source Name (DSN) in order to access your SQL Server, MySQL or Access database.

You could access ODBC DSN screen by clicking “ODBC DSNs” in the hosting space left navigation menu.

To create new data source click “Create ODBC DSN” button:

Specify data source name, select ODBC driver and then specify driver specific values.

Click “Save” to finish the wizard.
