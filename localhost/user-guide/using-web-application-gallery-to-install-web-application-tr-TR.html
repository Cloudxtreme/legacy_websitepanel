Web Uygulaması Kurmak için Web Uygulaması Galerisi Kullanma

Denetim masası, Microsoft Web Uygulama Galerisiyle tümleşiktir ve web sitenize popüler web uygulamalarını kolaylıkla kurmanıza olanak verir.

 

Web uygulamaları kataloğuna gözatmak için barındırma alanı sol gezinme menüsünde “Microsoft Web App Gallery”i tıklatın:

 

Kullanılabilir web uygulamalarının listesini almak için denetim masası doğrudan Microsoft web sitesine bağlanır. Uygulamalar listesi sürekli olarak büyümektedir; bizi takip edin.

 

Uygulamayı kurmak için listede yer alan simgesini veya “Kur” düğmesini tıklatın. Kurmak üzere olduğunuz uygulama hakkında ayrıntılı bilgiler verilecektir.

 

Bir barındırma planında bulunan kaynaklara bağlı olarak bazı uygulamalar kurulamayacaktır. Örneğin, SQL Server veritabanlarına barındırma alanında izin verilmiyorsa DotNetNuke veya BlogEngine gibi SQL Server veritabanı gerektiren uygulamaları kuramayacaksınız.

 

“İleri” düğmesini tıklattığınızda denetim masası, uygulama kurulum dosyalarını yüklemek için uygulama satıcı web sitesine bağlanacaktır (bu işlem biraz zaman alabilir).

 

Yükleme işlemi tamamlandıktan sonra uygulama kurulum ekranı karşınıza gelir:

 

Uygulamayı kurmak için web sitesini seçin. Uygulamayı web siteniz altındaki bir klasöre (sanal dizin) kurmak isterseniz adını girin; aksi takdirde, uygulama web sitesinin kök dizinine kurulacaktır.

 

Veritabanı seçeneklerini girin (gerektiğinde). Mevcut veritabanları ve veritabanı kullanıcılarından seçebilir veya sadece bu uygulama için yenilerini oluşturabilirsiniz.

 

“Veritabanı” bölümü altında diğer uygulama ayarları bulunuyor olabilir. Bu ayarlar, uygulamadan uygulamaya farklıdır.

 

Kurulum sihirbazını tamamlamak için “Kur” düğmesini tıklatın.

 

Başarısız bir kurulum durumunda daha fazla bilgi için hesabınız altındaki “Denetim Günlüğü” seçeneğini işaretleyebilirsiniz.
