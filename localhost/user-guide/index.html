WebsitePanel User Guide

Translations:

  • tr-TR

“User” role in WebsitePanel is the last security level in roles hierarchy. Users are created by reseller and they are consumers of hosting services. Users are able to create and manage domains, web sites, databases, files, mail accounts and other hosting resources.
