SQL Server Veritabanına Bağlanma

“Barındırma Alan Ana Sayfası”nı açın.

Barındırma alanınızdaki SQL Server veritabanlarının bağlantı bilgilerini görmek için “Alan Özet bilgileri” sayfasını açın ve “Veritabanları” bölümüne gidin:

 

Barındırma sağlayıcınız SQL Server'a harici bağlantıya izin veriyorsa bu bölümde sunucu adresini görmeniz gerekir. Microsoft SQL Server 2008 Management Studio Express kullanarak yerel makinenizdeki veritabanlarını yönetmek için bu adresi kullanın. Microsoft web sitesinden yüklenebilen ücretsiz bir üründür: http://www.microsoft.com/downloads/details.aspx?FamilyID=08e52ac2-1d62-45f6-9a4a-4b76a8564a2b&displaylang=en

 

“Veritabanları” bölümünde klasik ASP ve ASP.NET için bağlantı dizisi örnekleri de bulunmaktadır. www.connectionstrings.com web sitesinde daha fazla bağlantı dizisi bulabilirsiniz.
