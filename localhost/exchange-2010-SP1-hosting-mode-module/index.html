Exchange 2010 SP1 Hosting Mode Module Guide

Translations:

  • tr-TR

Exchange 2010 SP1 Hosting Mode Module works with Exchange 2010 SP1 hosting deployments (installed with the /hosting switch). WebsitePanel automates provisioning of Exchange organizations and setting quantitative quotas in control panel, as well as provides Exchange users with URL to Exchange Control Panel for managing their mailboxes.

  • Create Hosting Plans

  • Create Hosting Space and Organization
  • Updating Quotas
  • Changing Service Plan
  • Deleting Organization

For existing installations of Exchange 2007 or 2010 please refer to the guidance below on how to migrate your existing environment to Exchange 2010 SP1 hosting deployment.  Please note that migration steps may vary depending on Exchange configuration and look at this article as helpful guidance but not as a complete solution.
