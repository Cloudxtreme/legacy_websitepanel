Create Hosting Plans

Translations:

  • tr-TR

For each Exchange Service Plan admin or reseller will create Hosting Plan in WebsitePanel.

Go to Account Home – Hosting Plans and click “Create Hosting Plan” button.

Give Hosting Plan a meaningful name (for example the same name of for the Service that will be associated with it).

As Target Server, select Virtual Server associated with the same Service.

Check System and Exchange Hosting Mode quotas. For System quota limit number of domains (it should be equal or greater than number of domains for Exchange Hosting Mode quota). For Exchange Hosting Mode quota, enter quotas for Domains, Mailboxes, Contacts and Distribution lists. These quotas will override quotas in Exchange Service Plan.

Click “Save” button.
