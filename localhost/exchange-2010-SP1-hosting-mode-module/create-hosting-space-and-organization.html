Create Hosting Space and Organization

Translations:

  • tr-TR

Go to Admin Home – Customers and create a new Customer with user role.

For this customer, on their Account Home click “Create Hosting Space” button. Select hosting plan and click “Create”:

After hosting space is created, click <Hosting Space> link to access it and see left menu for available services there:

User will see the same menu when logged in to WebsitePanel. Click on “Exchange Hosting Mode” menu item – Create New Exchange 2010 Organization screen will open:

Enter Organization name, Administrator name and Password. For Password policy, please refer to this article.

Click “Create Organization”.

Note: in case user receives error message, to see information about the error, admin will need to go to Account Home – Audit Log and click on task link next to the error:

After organization is created, admin will see the following screen:

When logged in to WebsitePanel, customer will see similar screen but “Service Plan” section will be hidden.

To manage their mailboxes, contacts and distributions lists, users will click on ‘Exchange Control Panel’ link and access Exchange Control Panel. In WebsitePanel, users can update quotas, add/delete domains, send setup Instructions to specified e-mail address and delete organization.
