Managing Server System Processes

Translations:

  • tr-TR

WSP allows you to view and manage system processes running on the remote server. It may be useful in situations when you need to terminate some system process (say w3wp.exe process starting to consume much memory), but for some reason you don't want to login to the server using RDC or you can't to login.

To view the list of running system processes click "Configuration -> Servers" in the top menu. Then open server properties for which you want to view the processes and click "System Processes" link:

  

The following screen will appear displaying the list running system processes:

The screen is very similar to "Task Manager" Windows application.

Click "Terminate" button on appropriate row to terminate the process.

Click "Refresh" button to refresh the list of processes and their state.
