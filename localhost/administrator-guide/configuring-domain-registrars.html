Configuring Domain Registrars

Translations:

  • tr-TR

Here you can find options to configure domain registrars’ modules that help you to automate domains registration routine. There are three domain registrars currently supported: Enom, Directi and Offline Registrar.

Enom Registrar

Please find below description for each particular field on this screen.
Username - your Enom reseller account username you use to login on www.enom.com website;
Password - your Enom reseller account password you use to login on www.enom.com website;
Live Mode - if you wish to accept live domains registration tick this checkbox on. If you wish to perform demo domains registration leave it unchecked;
Once you specified all provider settings click “Save Settings” button.

Directi (ResellerClub) Registrar

Please find below description for each particular field on this screen.
Username – your reseller account username you use to login on ResellerClub website;
Password - your reseller account password you use to login on ResellerClub website;
Parent ID - your reseller’s account parent id on ResellerClub website;
Live Mode - if you wish to accept live domains registration tick this checkbox on. If you wish to perform demo domains registration leave it unchecked;
Secure Channel - tick this checkbox on if you wish to use SSL channel or tick checkbox off to use HTTP while sending data to the registrar's API;
Once you specified all provider settings click “Save Settings” button.
