Reporting

Translations:

  • tr-TR

Disk Space Report

Disk space report allows you to see how much of server disk space each customer's hosting space consumes.

You can click on the specified hosting space name and see disk space breakdown by resource groups (WEB, FTP, MAIL, etc.).

You can export the report in HTML format suitable for viewing in Excel.

Bandwidth Report

Bandwidth report allows you to see how much of bandwidth each customer's hosting space consumes for the certain period of time.

You can click on the specified hosting space name and see bandwidth breakdown by resource groups (WEB, FTP, MAIL, etc.).

You can export the report in HTML format suitable for viewing in Excel.

NOTE: The bandwidth value displayed on the left "Hosting Space" module is a summary bandwidth for the last 30 days.
