Quick Setup Outline

Translations:

  • tr-TR

If you want to setup WebsitePanel in minutes and start hosting your first customers you may just follow these quick steps:

  1. Setup System Settings (Setting up System Settings) 
  2. Add new remote server (Setting up Remote Servers)
  3. Add server IP addresses (Setting up Remote Servers)
  4. Add and configure server services (Setting up Server Services)
  5. Create a new virtual server (Virtual Servers)
  6. Add scheduled tasks for server bandwidth and disk space calculation (Scheduled Tasks)
  7. Change parking page, account summary, hosting space summary and password reminder letters (Account Settings)
  8. Create hosting plan for resellers or users (Hosting Plans)
  9. Create reseller or user account (User Accounts)
  10. Create a hosting space for reseller or user (Hosting Spaces)
