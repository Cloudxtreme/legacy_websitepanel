Creating Domain Name Products

Translations:

  • tr-TR

Here you can pull up your storefront with the domain names you would like to sell in conjunction with hosting plans. In order to create new domain name product click “Create Domain Product” button then you’ll be redirected to create domain name product screen:

Please find fields description below:
Domain TLD – top-level domain for the product; for example: com, net, org or us and etc.;
TLD Registrar – domain registrar to be used when register domain name;
Product SKU –product stock keeping unit code;
TLD Status - specifies domain name product availability to purchase;
Billing Cycles – available billing cycles that can be assigned to the product; select one appropriate to the product and click “Add Cycle” button once cycle item will appear in the list you need to specify setup fee, recurring fee and 2CO id (if applicable); Transfer Fee column can be left empty – it reserved for future use;
Once you’ve completed all fields on the screen click “Create Domain Product” button then you’ll be redirected to the Domain Names page.
In order to edit a particular domain name product please click on the product name in Domain Names page then you’ll be redirected to the edit domain name product screen.

Once you’ve completed all fields on the screen click “Save Changes” button to save changes. In case if you need to delete this product click “Delete” button.
