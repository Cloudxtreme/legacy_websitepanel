Sunucu Hizmetlerini Ayarlama

WebsitePanel'de uzak sunucu ekledikten sonra bunun hizmetlerini yapılandırmanız gerekir.

 

Hizmet Nedir?

"Hizmetler"i, WSP üzerinden kontrol edilmesi gereken ve son müşterilere sunulan, uzaktaki sunucuya kurulmuş bir uygulama olarak düşünün. Her bir hizmet sağlayıcı türüne (birlikte çalışabildiği uygulamanın adı/sürümü) göre belirlenir ve tüm hizmetler kaynak grupları (İS, WEB, FTP, POSTA, vb.) olarak gruplanır.

 

Sunucu Hizmetlerini Kaydetme

Sunucu hizmetleri eklemek için sunucu bilgileri formunu açın ve daha sonra "Hizmetler" bölümünü genişletin ("Yapılandırma -> Sunucular -> Sunucu bilgileri):

 

Sunucu hizmetini kaydetmeden önce ilgili uygulama sunucuda kurulmalı ve yapılandırılmalıdır.

Yeni bir hizmet kaydetmek için ilgili kaynak grubunda "Ekle" bağlantısını tıklatın.

Aşağıdaki ekran görüntülenecektir:

 

Hizmetin ekran adını belirtin ve hizmet sağlayıcıyı seçin. "Hizmet sağlayıcı" açılır listesindeki değerler, seçili kaynak grubuna göre değişir. Kurulu uygulamaya karşılık gelen sağlayıcıyı seçin.

 

Sunucuyu eklemek için "Hizmet Ekle" düğmesini tıklatın. Bundan sonra hizmet özellikleri ekranı görüntülenecektir: Bunun görünümü, önceki adımda seçilen hizmet sağlayıcısına göre değişir. Aşağıdaki alt bölümler, tüm hizmet sağlayıcıları için sunucu ayarlarını anlatmaktadır:

·         Windows Server 2003/2008

 

·         Web

·         Microsoft IIS 7.0

·         Microsoft IIS 6.0

 

·         FTP

·         MS FTP 7.0

·         MS FTP 6.0

·         Gene6 FTP

 

·         Posta

·         SmarterMail

·         MailEnable

·         MDaemon

·         Merak

·         hMailServer

·         Ability Mail Server

·         ArgoMail Server

 

·         SQL Server 2000/2005/2008

·         SQL Server 2000/2005/2008

 

·         MySQL 4/5

·         MySQL Server 4/5

 

·         DNS

·         MS DNS
