Setting up System Settings

Translations:

  • tr-TR

Right after installing WebsitePanel you should setup WebsitePanel system settings. They need to be correctly setup to send e-mail messages and work with backup/restore module.

In order to access System Settings page login under "serveradmin" and navigate to "Configuration -> System Settings" page.

SMTP Server Settings

Specify correct SMTP server settings to allow WebsitePanel send e-mail messages. Please, keep in mind that the application that sends e-mail messages is WebsitePanel Enterprise Server, not Portal.

  • Server - IP or hostname of mail server. If you want to send messages through SmarterMail then make sure it is an IP that bound to SmarterMail mail domain.

  • Port - SMTP port to use. By default it is 25.

  • User and Password - if you have SMTP relays disabled on mail server you should specify username and password of the account that has rights to send e-mail messages.

  • Enable SSL - if your SMTP is configured to accept only SSL client connections, tick this checkbox.

Backup/Restore Settings

  • Backup Folder Path - physical path to the folder where temporary backup files will be placed. Make sure the account of Enterprise Server application pool has write access to this folder. The pool of WebsitePanel Enterprise Server works under "WSPEnterpriseServer" account, so this account should have "write" NTFS access to backups temporary folder.
