Detaching Space Items

Translations:

  • tr-TR

WebsitePanel can not exclude Windows and other 3rd-party management tools that can be used to change server environment. Sometimes WSP meta-base and live environment become unsynchronized, e.g. web site was deleted from IIS admin tool or mail account was deleted from web mail management interface. So, opening deleted web site or mail account or database throws an error in WSP.

How to remove those "broken" items from WSP meta-base? Administrators can use "Detach" function to remove "broken" or non-existing items from hosting spaces.

"Detach" button is available to administrators on any space items list screen:

"Detach" function removes the information about item from WSP meta-base only, but does not delete the item physically.
