Creating Categories

Translations:

  • tr-TR

Expand "Ecommerce Admin" menu on the left and click "Categories" menu item to see the list of categories:

Click ‘Create Category’ button to create a new category:

Common Fields

Category Name - here you can enter category name that will appear in storefront catalogue.

Category SKU - unique stock keeping unit code for the category. Enter here abbreviated category code.

Parent Category - list of categories that can be assigned as parent for the category you're creating. If this category is root then select "Select parent..." first item in the list. You can use this list to move category from one parent to another when editing category.

Short Description - short category description text

Full Description - full category description text

Once you've complete all fields in this screen click "Create Category" button to add new category.

How To Edit Category

If you need to edit category expand "Ecommerce Admin" menu and click on the "Categories" menu item. After "Categories" screen will appear browse to the category you wish to edit and click on the pencil icon on the left side then "Edit Category" screen will appear. Do the necessary changes and click "Update" button to update category.

How To Delete Category

If you need to delete category expand "Ecommerce Admin" menu and click "Categories" menu item. After "Categories" screen will appear browse to the category you wish to delete and click on the pencil icon on the left side then "Edit Category" screen will appear. Click "Delete" button and the category will be removed immediately. Please note that if you are deleting a category that has sub-categories then all of them become categories without parent (root categories).
