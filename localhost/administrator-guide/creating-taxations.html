Creating Taxations

Translations:

  • tr-TR

Here you can create taxation rules reflecting your specific business needs. In order to create new taxation rule click “Add Tax” button then you’ll be redirected to add taxation rule screen.

Please find fields description below:
Description – taxation rule description that can be displayed when rendering invoice template;
Tax Status - specifies taxation rule availability to be applied;
Country – country that taxation rule applies to;
State/Province – country state or province that taxation rule applies to;
Tax Type – taxation rule type (Fixed amount or Percentage/Progressive amount);
Amount – taxation rule amount if applicable; can be also 0;
Once you’ve completed all fields on the screen click “Save Tax” button then you’ll be redirected to the Taxations page.
In order to edit a particular taxation rule please click on the rule name in Taxations page then you’ll be redirected to the edit taxation rule screen.

Once you’ve completed all fields on the screen click “Save Tax” button to save changes. In case if you need to delete this taxation rule click “Delete Tax” button.
