Managing Server RDC Connections

Translations:

  • tr-TR

By default dedicated Windows server allows a limited number of active RDC connections and if you work in a group on a single server sometimes it's hard to login as new RDC connection will be refused. WSP allows you to view active RDC sessions on some server and if required to reset selected session (for example, in case person working on the session forgot to log off and just closed RDC window).

To view the list of active server RDC (Remote Desktop Connection) sessions click "Configuration -> Servers" in the top menu. Then open server properties for which you want to view RDC connections and click "Remote Desktop Sessions" link:

  

The following screen will appear displaying the list of active RDC connections:

To reset specified RDC session click on "Reset" icon against its line.
