What is Peer Account?

Peer account is an alias for its parent user account. Peer accounts are helpful when there is a need for several administrators to manage hosting spaces in the same hosting account.

Creating a Peer Account

To see the list of user peer accounts click "Peers" link in the left "User Account Menu". The following screen will appear:

To add a new peer account click "Add Peer" button. You'll be redirected to "Peer Account Details" screen shown below:

Fill form fields and click "Add" to create peer account.
