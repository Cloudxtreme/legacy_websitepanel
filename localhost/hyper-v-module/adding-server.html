Adding Server

Translations:

  • tr-TR

Login into WebsitePanel under "serveradmin" account (default password is "serveradmin") and open "Configuration -> Servers" page.

Register WSP Server installed on Hyper-V virtualization server by clicking "Add Server" button.

Specify "Server URL" and "Server password" to the correct values selected during WSP Server installation.

Uncheck "Search installed software" checkbox.

After WSP Server is successfully added you will be redirected to Server properties page.

For more information on addition servers, read this article.
