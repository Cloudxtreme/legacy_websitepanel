Setup Networking

Translations:

  • tr-TR

If customer VPS should be connected to External network there should be at least one "External" virtual network configured in Hyper-V. Default virtual network with the name "network adapter - Virtual Network" is created during Hyper-V role installation. If such network does not exist or you want to create another network bound to different network adapter (for example “Management network”) you can do it in "Virtual Network Manager" of Hyper-V Manager snap-in:

Later you will be selecting this network from the list to setup Hyper-V module in WSP.
