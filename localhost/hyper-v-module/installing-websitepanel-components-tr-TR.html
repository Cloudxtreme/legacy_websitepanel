WebsitePanel VPS Yapılandırmaları

Tek Sunucu Hyper-V Çözümü Uygulaması

 

·         En basit Hyper-V çözümü uygulaması, Hyper-V ana bilgisayarı üzerinde Denetim Masası kurulumudur

·         “Web Sunucusu” rolü de ana bilgisayar WebsitePanel bileşenlerine kurulmalıdır

·         SQL Server Express (veya üzeri) uygulaması da ana bilgisayar WebsitePanel veritabanına kurulmalıdır

 

Yerel Yönetim Hyper-V Çözümü Uygulaması

 

Avantajları:

·         Active Directory gerekli değildir

·         Yönetim sunucusu, Windows Server 2003 veya 2008 üzerine kurulabilir

 

Dezavantajları:

·         Hyper-V ana bilgisayarı Windows Server 2008 “Tam Kurulum” olmalı, “Sunucu Çekirdeği” veya “Hyper-V Sunucu” olmamalıdır

·         WSP Sunucu uygulaması Hyper-V ana bilgisayarında kurulu olmalıdır

·         Hyper-V ana bilgisayarı için WSP Uzak Sunucu operatörü gereklidir

·         “Web Sunucusu” rolü WSP Uzak Sunucu operatörünün barındırılacağı Hyper-V bilgisayarda kurulu olmalıdır

 

Merkezi Yönetim Hyper-V Çözümü Uygulaması

 

Avantajları:

·         Hyper-V ana bilgisayarı Windows Server 2008 “Sunucu Çekirdeği” veya “Hyper-V Sunucu 2008” olabilir

·         Tüm Hyper-V ana bilgisayarları, “Yönetim Sunucusu” üzerinde kurulu tek bir WSP Sunucudan yönetilir

·         Müşteri iş yükleri, maksimum Hyper-V ana bilgisayar kaynağı kullanır

·         WSP Sunucu, Hyper-V sunucu üzerinde kurulu değildir ve WSP Uzak Sunucu operatörü Hyper-V ana bilgisayarı için gerekli değildir

 

Dezavantajları:

·         “Yönetim Sunucusu” yalnızca Windows Server 2008 olmalıdır

·         Active Directory gereklidir
