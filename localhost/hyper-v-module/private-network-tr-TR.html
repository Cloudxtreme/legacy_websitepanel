IP Adresleri Biçimi

Özel – ilk IP adresi ve ağ maskesi belirlenmesini mümkün kılar.

192.168.0.1/16 - 255.255.0.0 ağ maskesi ile 192.168.0.1 değerinden başlayan IP adresleri oluşturur;

172.16.0.1/12 - 255.240.0.0 ağ maskesi ile 172.16.0.1 değerinden başlayan IP adresleri oluşturur;

10.0.0.1/8 - 255.0.0.0 ağ maskesi ile 10.0.0.1 değerinden başlayan IP adresleri oluşturur

IP Adresi / CIDR

Özel ilk IP adresi ve al ağ maskesi belirlenmesini mümkün kılar.

Varsayılan Ağ Geçidi

TCP/IP protokolü özellikleri “Varsayılan ağ geçidi” alanında ayarlanacak olan IP adresi.

Tercih Edilen Ad Sunucusu

TCP/IP protokolü özellikleri “Tercih edilen ad sunucusu” alanında ayarlanacak olan IP adresi.

Alternatif Ad Sunucusu

TCP/IP protokolü özellikleri “Alternatif ad sunucusu” alanında ayarlanacak olan IP adresi.
