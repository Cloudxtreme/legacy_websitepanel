Single Server Hyper-V Solution Deployment

  • The simplest Hyper-V solution deployment where Control Panel installed on Hyper-V host
  • “Web Server” role must be also installed to host WebsitePanel components
  • SQL Server Express (or above) must be installed to host WebsitePanel database

Local Management Hyper-V Solution Deployment

Pros:

  • Active Directory is not required
  • Management server could be installed either on Windows Server 2003 or 2008

Cons:

  • Hyper-V host must be Windows Server 2008 “Full Installation”, but not “Server Core” or “Hyper-V Server”
  • WSP Server application must be installed on Hyper-V host
  • WSP Remote Server agent is required for Hyper-V host
  • “Web Server” role must be installed on Hyper-V machine to host WSP Remote Server agent

Centralized Management Hyper-V Solution Deployment

Pros:

  • Hyper-V host could be Windows Server 2008 “Server Core” or “Hyper-V Server 2008”
  • All Hyper-V hosts are managed from a single WSP Server installed on “Management Server”
  • Customer workloads use maximum Hyper-V host resources available
  • Neither WSP Server is installed on Hyper-V server nor WSP Remote Server agent is required for Hyper-V host

Cons:

  • “Management Server” must be Windows Server 2008 only
  • Active Directory is required
