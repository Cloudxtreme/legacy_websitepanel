WSP Sanal Bilgisayar Yapılandırma Hizmeti Kurma

WebsitePanel Sanal Bilgisayar Yapılandırma Hizmeti (WSP VmConfig), misafir işletim sisteminde çalışan ve aşağıdaki işlevlere sahip bir Windows Hizmetidir:

·         Misafir işletim sisteminin yapılandırılması için denetim modüllerini yürütür.

·         WebsitePanel kullanıcı arabiriminde (UI) görüntülenen gerçek zamanlı istatistikler alır (RAM, HDD).

 

WSP VmConfig, şu denetim modüllerini içerir:

·         Bilgisayar adı değiştirme

·         Yönetici şifresi değiştirme

·         Ağ adaptörü yapılandırması değiştirme

 

WSP VmConfig hizmeti, “Hyper-V Data Exchange” tümleştirme bileşeni aracılığıyla Hyper-V ana bilgisayarı ile iletişim kurar; bu nedenle hizmetin çalışır durumda olduğundan emin olun.

 

Sistem gereksinimleri:

  • Windows XP tüm sürümleri, Windows Vista tüm sürümleri, Windows Server 2003 tüm sürümleri, Windows Server 2008 tüm sürümleri
  • NET Framework 2.0 çalışma zamanı:

·         Microsoft .NET Framework Version 2.0 Yeniden Dağıtılabilir Paket (x86)

·         Microsoft .NET Framework Version 2.0 Yeniden Dağıtılabilir Paket (x64)

 

Not: OS ile birlikte varsayılan olarak kurulduğundan Windows Server 2008 üzerinde .NET Framework 2.0 kurmanız gerekli değildir.

 

Kurulum talimatları:

  • En yeni WSP VmConfig sürümünü SourceFourge'dan yükleyin: http://sourceforge.net/projects/websitepanel/files/
  • Sıkıştırılmış dosya içeriğini hedef klasöre çıkarın; örn. “C:\Program Files\WebsitePanel\VmConfig”.
  • Kurulum toplu işini kullanarak VmConfig hizmetini kurun:

·         Windows Explorer uygulamasında WSP.VmConfig.exe çalıştırılabilir dosyasının bulunduğu hedef dizindir.

·         “install.bat” dosyasını çalıştırın.

  • “Başlat -> Yönetim Araçları -> Hizmetler” öğesini tıklatarak “Hizmetler” eklentisini açın. Listede “WebsitePanel Sanal Bilgisayar Yapılandırma Hizmeti” görüntülenmelidir.
  • “WebsitePanel Sanal Bilgisayar Yapılandırma Hizmetini” başlatın.
  • WSP.VmConfig.log dosyasını açın ve hata içermediğinden emin olun. Normal koşullarda hizmet başladıktan hemen sonra şu metni içermelidir (X.X.X.X aracın sürümünü ifade eder):
  • [11/26/2010 4:53:32 PM] APP: WSP.VmConfig X.X.X.X başarıyla başlatıldı
  • [11/26/2010 4:53:32 PM] START: Denetim modülleri yükleniyor...
  • [11/26/2010 4:53:32 PM] END: 3 modül yüklendi
