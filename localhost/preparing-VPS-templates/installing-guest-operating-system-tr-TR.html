Misafir İşletim Sistemi Kurma

Yeni oluşturulan ana VM, Windows Server 2008 kurulum DVD'sinden yüklenir ve OS kurulum işlemi başlar:

Fare desteği için Tümleştirme Hizmetlerinin misafir OS'de kurulması gerektiğinden farenin bu durumda kullanılamayacağını unutmayın.

Listeden Windows Server 2008 Edition seçin ve “İleri” düğmesini tıklatın. Örneğimizde “Windows Server 2008 Standard (Tam Kurulum) x64” kurulmaktadır.

Windows Server 2008 kurmak için sihirbazdaki diğer adımları uygulayın:

Kurulum sırasında, her yeniden başlatmada VM “CD veya DVD'den yüklemek için herhangi bir tuşa basın...” iletisi verir; mesajı dikkate almayın ve HDD'den yüklemeye devam edin.
