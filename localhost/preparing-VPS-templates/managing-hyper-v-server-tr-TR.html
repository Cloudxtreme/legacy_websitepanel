Hyper-V Sunucuyu Yönetme

Hyper-V rolü kurulu tam Windows Server 2008 üzerinde VHD ve sanal bilgisayar ile çalışmak daha uygundur.

 

Ancak uzak Hyper-V Server 2008 veya Windows Server 2008 Sunucu Çekirdeği kurulu ise “Windows Vista için Hyper-V Uzaktan Yönetim Güncellemesini (KB952627)” kullanabilirsiniz: http://www.microsoft.com/downloads/details.aspx?FamilyID=BF909242-2125-4D06-A968-C8A3D75FF2AA&displaylang=en

 

64-bit Vista için:

http://www.microsoft.com/downloads/details.aspx?familyid=88208468-0AD6-47DE-8580-085CBA42C0C2&displaylang=en

 

Bu güncelleme ile ilgili önemli notlar:

1. Windows Vista Home Basic veya Premium sürümlerinde çalışmaz.

2. “Yönetim Araçları” menü öğesi etkinleştirilmelidir:

 

“Yönetim Araçları” menüsünü etkinleştirmek için Görev Çubuğunu sağ tıklatın -> “Özellikler” menü öğesi -> “Başlat menüsü” sekmesi -> “Özelleştir...” düğmesini tıklatın.

 

Tam Windows Server 2008 uzaktan yönetim için “Windows Vista için Microsoft Uzak Sunucu Yönetim Araçlarını (KB941314)” kullanabilirsiniz: http://www.microsoft.com/downloads/details.aspx?FamilyID=9ff6e897-23ce-4a36-b7fc-d52065de9960&displaylang=en

 

64-bit Vista için:

http://www.microsoft.com/downloads/details.aspx?familyid=D647A60B-63FD-4AC5-9243-BD3C497D2BC5&displaylang=en

 

Önemli notlar:

1. Windows Vista Business sürümü veya daha yukarısı gereklidir

2. Service Pack 1 kurulu olmalıdır.

Tüm örneklerimizde Hyper-V rolünün kurulu olduğu tam Windows kurulumu kullanılacaktır.
