Windows 2003 Web Server'ı Yapılandırma

PHP Uzantılarını, IIS'te Etkinleştirme

Web PI kullanarak PHP kurduktan sonra PHP uzantısını IIS'te etkinleştirmelisiniz. IIS Yöneticisinde sol gezinme ağacında Web Hizmeti Uzantıları'nı sağ tıklatın ve daha sonra Yeni web hizmeti uzantısı ekleyin öğesini seçin.

 

Uzantı adını "PHP 5" olarak belirtin, php-cgi.exe dosyasının yolunu seçin ve daha sonra Uzantı durumunu İzin Veriliyor olarak ayarlayın.

 

Tamam öğesini tıklatın.

 

IIS'te, FTP ve SMTP Hizmetlerini kurma

Sunucu üzerinde dosyalar ve klasörlerle çalışmak amacıyla FTP istemcisini kullanmak için IIS 6'nın bir parçası olan FTP Hizmetini kurmanız gerekir. WebsitePanel'den bildirimler göndermek için SMTP hizmetini de kurmanız gerekir.

 

Windows Denetim Masasında Program Ekle/Kaldır öğesini seçin ve daha sonra Windows Bileşenleri Ekle/Kaldır öğesini tıklatın. Uygulama Sunucusu için Ayrıntılar'ı tıklatın ve daha sonra Internet Information Services (IIS) öğesi için tekrar Ayrıntılar'ı tıklatın. Dosya Aktarım Protokolü (FTP) Hizmeti'ni ve SMTP Hizmeti öğelerini seçin ve daha sonra Tamam öğesini tıklatın.

 

IIS Günlük Kaydını Yapılandırma

WebsitePanel'de web sitesi trafiğini toplamak ve analiz etmek için IIS'te günlük kaydını etkinleştirmeniz gerekir. Varsayılan olarak doğru biçimde oturum açmak daha önceden etkinleştirilmiştir ancak bazı önemli seçenekler etkinleştirilmemiştir.

 

IIS Yöneticisinde Web Siteleri düğümünü sağ tıklatın ve daha sonra Özellikler'i seçin. Günlük kaydını etkinleştir bölümünde Özellikler'i tıklatın ve daha sonra Gelişmiş sekmesini tıklatın. Gönderilen bayt ve Alınan bayt öğelerini seçin.

 

Tamam öğesini tıklatın.

 

Şifre Karmaşıklık Düzeyi İlkesini Devre Dışı Bırakma

Şifre karmaşıklık düzeyi ilkesini devre dışı bırakmak için Yerel Güvenlik Ayarları yöneticisini açın (Başlat > Yönetim Araçları), Hesap İlkeleri'ni genişletin ve daha sonra Şifre İlkesi'ni genişletin. Ayarlanan şifre, karmaşıklık düzeyi gereksinimlerini karşılamalıdır ilkesini Devre dışı olarak ayarlayın.
