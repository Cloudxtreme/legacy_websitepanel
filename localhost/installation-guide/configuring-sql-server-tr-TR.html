SQL Server'ı Yapılandırma

Bu konu, SQL Server'ın bir örneğinin WebsitePanel ile birlikte çalışacak şekilde nasıl yapılandırılacağını anlatmaktadır. WebsitePanel üzerinden Windows Kimlik Denetimi kullanarak (SQL Server örneğine erişmek için hesap kimlik bilgileri WebSitePanel veritabanında depolanmaz) SQL Server'ı yönetmek için SQL Server'daki sysadmin görevine bir Windows Yöneticiler grubu eklemeniz gerekir.

 

Sunucu Yöneticilerine "Yöneticiler" grubunu ekleme

Not  WSPServer hesabı, WSP Kurulum Aracı tarafından sysadmin grubuna otomatik olarak ekleneceği için WebSitePanel Bağımsız Kurulum paketini kurmak için bu adımı atlayabilirsiniz.

 

SQL Server Management Studio'yu açın, Güvenlik öğesini sağ tıklatın, Oturum Açma düğümünü seçin ve daha sonra Yeni Oturum'u seçin. Yeni Oturum iletişim kutusunda Oturum açma adı için "BUILTIN\Administrators" girin.

 

Sunucu Görevleri sekmesine geçin, sysadmin görevini seçin ve daha sonra Tamam öğesini tıklatın.

 

Yeni grup eklenir.

 

SQL Server'ı Harici Bağlantıları Kabul Edecek Şekilde Yapılandırma

Sunucu üzerinde kurulu SQL Server Express'in TCP/IP üzerinden harici bağlantıları kabul edecek şekilde etkinleştirmek için aşağıdaki Bilgi Bankası makalesindeki talimatları uygulayın:

http://support.microsoft.com/kb/914277

 

Bu makale, SQL Server 2005 ve SQL Server 2008 için geçerlidir.
