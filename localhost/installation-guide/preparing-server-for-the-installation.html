Preparing Server for the Installation of WebsitePanel Components

Translations:

  • tr-TR

In case of Multi-Server Setup you will need to install WebsitePanel components one-by-one on your servers. Unlike WebsitePanel Standalone Setup that can be done using Microsoft Web Platform Installer, WebsitePanel components have to be installed with WebsitePanel Installer and servers need to be correctly configured prior to the installation. The configuration process is different for Windows Server 2003 and for Windows Server 2008. The easient way to prepare servers for the installation is to use Microsoft Web Platform Installer (Web PI). Web PI enables you to easily install and configure the following products:

  • Internet Information Services
  • .NET Framework 2.0
  • .NET Framework 3.5 SP1
  • SQL Server 2008 with SQL Management Studio Express
  • PHP 5.x
  • Additional libraries and programs

For more details on which products to install, please refer to the following articles:

Prepare Windows Server 2003 for WebsitePanel Installation

Prepare Windows Server 2008 for WebsitePanel Installation

You can download Web PI from the following location:

http://www.microsoft.com/web/downloads/platform.aspx

Note   On Windows Server 2003, the installation process might ask for your permission in order to install .NET Framework 2.0. Select the option to allow this installation.
