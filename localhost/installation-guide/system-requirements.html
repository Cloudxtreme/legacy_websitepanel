System Requirements

Hardware Requirements

Dedicated Server

In order to install WebsitePanel and make your hosting business you must have a dedicated Web server. We don't dictate specific requirements for your hardware; a typical server with 1GB RAM and a 40GB hard drive is adequate.

You must have administrative access (either local or via Remote Desktop) to manage the server, including permissions to install and configure server software, check system logs, etc.

Dedicated IP Addresses

If you are planning to host and manage DNS and mail software on your server, you should have at least two dedicated IP addresses that point to your server. In case you install and configure everything using only one IP address you'll get incorrect DNS zones and mail exchangers.

Internet Access

Your dedicated server must be connected to the Internet for you to be able to install and configure WebsitePanel components.

Software Requirements

Operating System

The following operating systems are supported:

  • Windows Server 2003
  • Windows Server 2003 R2
  • Windows Server 2008
  • Windows Server 2008 R2

Any edition of operating systems listed here is supported. Both 32-bit and 64-bit editions are supported.

Web Server

Microsoft Internet Information Services (IIS) version 6 or 7 must be installed in order to host WebsitePanel components.

Database Server

SQL Server is required in order to host the internal WebsitePanel database. The WebsitePanel database can be installed on the following versions of SQL Server:

  • SQL Server 2005 Express edition or higher
  • SQL Server 2008 Express edition or higher

SQL Server Express is free. You can download it from the following location:

http://www.microsoft.com/express/sql/download/

Virtualized Environments

WebsitePanel can be installed on a dedicated physical server or on a virtual machine hosted in one of the following virtualization environments:

  • Microsoft Hyper-V
  • Microsoft Virtual PC
  • Virtuozzo
  • VmWare
  • XenServer
