WebsitePanel Nasıl Çalışır

WebsitePanel, Windows barındırması için web siteleri, FTP hesapları, veritabanları veya birden fazla sunucuda kurulabilen diğer barındırma kaynaklarını yönetmeye izin veren bir denetim masasıdır. WebsitePanel ayrıca, yetkili satıcılar için boş etiketli ürünler ve son kullanıcılar için self servis denetim masası sağlamaktadır.

 

WSP Mimarisi

WebsitePanel, dağıtık bir web tabanlı uygulamadır. Üç bileşenden oluşmaktadır:

 

·         WSP Portalı

·         WSP Enterprise Server

·         WSP Sunucusu

 

Barındırma Ortamı

 

Tüm üç bileşen de ya IIS sürüm 6 ya da IIS sürüm 7 kullanılarak barındırılması gereken web uygulamalarıdır.

WSP Portalı

 

WSP Portalı, WebsitePanel'in kullanıcı arabirimi (sunum katmanı) bölümüdür. Herhangi bir iş mantığı içermeyen bir ASP.NET uygulamasıdır. Bunun yerine, çeşitli işlemleri gerçekleştirmek için SOAP kullanarak WSP Enterprise Server ile iletişim kurar. Sunum katmanı, tüm UI öğelerini içerir. Düğme tıklatma gibi kullanıcı hareketleri, WSP Enterprise Server'da web hizmeti yöntemlerini başlatır.

 

WSP Portalı web uygulamasına harici tüketicilerden erişilebilir. Sınırlı ayrıcalığı bulunan bir hesapla ilişkili bir IIS uygulama havuzu içinde çalışır.

 

WSP Enterprise Server

WSP Enterprise Server, uygulamanın iş mantığını oluşturan temel WSP bileşenidir. WSP Enterprise Server, kullanıcı hesapları, barındırma planları, barındırılan kaynaklar hakkında bilgi ve hizmet ayarları dahil dahili verileri depolamak için bir SQL Server veritabanı kullanır.

 

WebsitePanel'deki her işlem, .NET Framework, PHP, Java ve Perl dahil herhangi bir programlama ortamından erişilebilen bir XML web hizmeti olarak kullanılabilir. Bu nedenle, WebsitePanel web hizmetleri, üçüncü taraf uygulamalar tarafından tümleştirme için kullanılabilir.

 

Barındırma ortamını yönetmek için (web siteleri, FTP ve posta hesapları, veritabanları oluşturmak, vb.) WSP Enterprise Server, XML web hizmetlerini kullanarak WSP Server ile iletişim kurar.

 

WSP Enterprise Server, herhangi bir UI sunmaz ve sadece web hizmetleri sağlar. Genelde barındırma ortamınızdaki bilgisayarlar tarafından kullanılabilir; bu uygulama sadece dahili IP adreslerini veya dahili geri döngü adresi olan 127.0.0.1 adresini dinler. WSP Enterprise Server, sınırlı ayrıcalığı bulunan bir hesapla ilişkili bir uygulama havuzu içinde çalışır.


WSP Sunucusu

WSP Sunucusu, hedef sunucularda dosyalar ve klasörle çalışma bunların izinleri, e-posta, veritabanları, IIS, vb. gibi düşük düzeyli yönetim işlemleri gerçekleştiren web uygulamasıdır

 

WSP Sunucusu, bir uzak sunucu aracısı olarak düşünülebilir ve WSP denetim masasında yönetilmesi gereken barındırma ortamındaki her sunucuya kurulmalıdır. WSP Sunucusu, yönetim ayrıcalıklarına sahip bir uygulama havuzunda çalışır (yani, AD'nin de yönetilmesi gerekiyorsa uygulama havuzunun kimliği, yerel Yöneticiler grubu veya Etki Alanı Yöneticileri grubunun bir üyesi olmalıdır).
