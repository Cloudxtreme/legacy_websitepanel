WebsitePanel'i Web Platform Installer ile Kurma

WebsitePanel Bağımsız Kurulumu, Microsoft Web Platform Installer (Web PI) kullanılarak kolay bir şekilde gerçekleştirilebilir.

Web PI sunucunuzda zaten kuruluysa açın ve Ürünler -> Sunucu öğesini seçin.

 

WebsitePanel'in yanındaki Ekle düğmesini ve daha sonra da Kur düğmesini tıklatın. Sistem, kurulum için önerilen uygulamalar listesini sunacaktır. WebsitePanel kurulumu, sunucu üzerinde .NET Framework 2.0, ASP.NET ve SQL Server Express kurulu olmasını gerektirir. Bu ürünlerden herhangi birinin kurulu olmaması durumunda WebsitePanel ile birlikte listede gösterilecektir:

 

Kabul Ediyorum düğmesini tıklattığınızda kurulum başlatılacaktır.

SQL Server Express kuruluyorsa kimlik denetim modunu seçmeniz ve SQL Server yönetici şifresini girmeniz istenecektir. Karma Mod Kimlik Denetimi (varsayılan seçim) öğesini seçin, Şifre ve Şifreyi Onayla alanlarını doldurun ve Devam seçeneğini tıklatın.

 

Başarılı kurulum tamamlandığında onay ekranı görüntülenir:

 

Masaüstünüzde iki simge oluşturulacaktır: WebsitePanel Kurulum Aracı ve WebsitePanel'de Oturum Aç. Gelecekte WebsitePanel Kurulum Aracını WebsitePanel'i yeni sürümlere yükseltmek için kullanacaksınız.

Denetim masasında ilk kez oturum açmak için WebsitePanel'de Oturum Aç simgesini tıklatın. serveradmin ve admin hesapları için şifre girmenizin isteneceği WebsitePanel Hesaplarını Ayarlama sayfası ekrana gelecektir:

 

Her iki hesap için de Şifre Girme ve Şifre Onaylama alanlarını doldurun ve Kurulumu Tamamla düğmesini tıklatın. admin hesabıyla WebsitePanel'de oturum açılacak ve hesap ana sayfası ekrana gelecektir:

 

WebsitePanel kurulumu tamamlanmıştır. Sunucularınızı WebsitePanel ile çalışacak şekilde hazırlamak için Kurulum Kılavuzunun Sunucuları Yapılandırma bölümünü uygulayın.

 

Not WebsitePanel, sadece Bağımsız Kurulum için Web PI ile birlikte kurulabilir. Ayrı WebsitePanel bileşenleri kurmak için WebsitePanel bileşenlerini kurmak için Sunucuları Hazırlama ve WebsitePanel Kurulum Aracı makalelerindeki talimatları uygulayın.
