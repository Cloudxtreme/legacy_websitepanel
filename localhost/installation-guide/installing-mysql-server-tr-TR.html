MySQL Server Kurma

PHP'de yazılmış birçok uygulama, bir MySQL veritabanı sunucusu gerektirir. Ücretsiz MySQL Community Server'ı aşağıdaki konumdan yükleyebilirsiniz:

 

http://dev.mysql.com/downloads/mysql/5.1.html

 

Platformunuzu (Windows, 32-bit veya 64-bit sürümü) seçin ve daha sonra Windows Essentials paketini yükleyin. (Genelde, seçili platform altındaki en küçük olan pakettir.)

 

Kurulum sihirbazını çalıştırın ve Kurulum Türü olarak Tipik seçeneğini seçin.

 

Kurulum sonrasında MySQL hizmet yapılandırmasına gidin.

 

Standart Yapılandırma'yı seçin ve daha sonra İleri'yi tıklatın.

 

MySQL Windows hizmetinin ayarlarını belirtin.

 

Kök hesabı için bir şifre belirtin ve Uzak makinelerden kök erişimini etkinleştir öğesini seçin. Bazı durumlarda bu, uygulamalarınız için MySQL yapılandırmanızı kolaylaştırır. İleri'yi tıklatın.

 

Sonraki adım için Yürüt seçeneğini tıklatın. Tüm yapılandırma adımları başarıyla tamamlanmalıdır.

 

MySQL hizmetini test etmek için bir komut dosyasını açın, MySQL kurulum klasörüne gidin ve bazı MySQL komutlarını çalıştırın (kullanılabilir veritabanlarının listesini görüntüleme gibi). Aşağıdaki çizim, MySQL'in nasıl test edileceğine bir örnek vermektedir.
