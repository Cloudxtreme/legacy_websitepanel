Installing WebsitePanel with Web Platform Installer

Translations:

  • tr-TR

WebsitePanel Standalone Setup can be easily installed using Microsoft Web Platform Installer (Web PI).

If Web PI is already installed on your server, open it and select Products -> Server.

Click Add button next to WebsitePanel and then click Install button. System will present the list of applications proposed for the installation. WebsitePanel installation requires .NET Framework 4.0, ASP.NET and Microsoft SQL Server Express to be installed on the server. In case any of these products are not installed, they will be shown in the list together with WebsitePanel:

Click I Accept button and installation will start.

If SQL Server Express is being installed than you will be asked to choose authentication mode and enter SQL Server administrator password. Select Mixed Mode Authentication (default selection), enter Password and Confirm Password fields and click Continue:

After the installation is complete you will be presented with the confirmation screen:

Two icons will be created on your desktop: WebsitePanel Installer and Login to WebsitePanel. In the future you would use WebsitePanel Installer to upgrade WebsitePanel to new versions.

Click on the Login to WebsitePanel icon to do you first login to a control panel. You will be presented with Setup WebsitePanel Accounts page where you will be offered to enter passwords for serveradmin and admin accounts:

Enter Password and Confirm password fields for both accounts and click Complete Setup button. You will be logged in to WebsitePanel under admin account and presented with the account home page:

WebsitePanel installation is complete. Follow the Configuring Servers section of Installation Guide to prepare your servers to work with WebsitePanel.

Note WebsitePanel is only installable with Web PI for Standalone Setup. For installing separate WebsitePanel components please follow the Preparing Servers for installing WebsitePanel components and WebsitePanel Installer articles.
