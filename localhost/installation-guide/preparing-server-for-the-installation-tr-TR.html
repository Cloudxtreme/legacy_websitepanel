WebsitePanel Bileşenlerinin Kurulumu için Sunucuyu Hazırlama

Çoklu Sunucu Kurulumu söz konusu olduğunda WebsitePanel bileşenlerini sunucularınıza tek tek kurmanız gerekir. Microsoft Web Platform Installer kullanarak gerçekleştirilebilecek WebsitePanel Bağımsız Kurulumundan farklı olarak WebsitePanel bileşenlerinin WebsitePanel Kurulum Aracı ile kurulması ve sunucuların kurulum öncesinde düzgün şekilde yapılandırılması gerekir. Yapılandırma işlemi, Windows Server 2003 ve Windows Server 2008 için farklıdır. Sunucuları kurulum için hazırlamanın en kolay yolu, Microsoft Web Platform Installer'ı (Web PI) kullanmaktır. Web PI, aşağıdaki ürünleri kolay bir şekilde kurmanızı ve yapılandırmanızı sağlar:

·         Internet Information Services

·         .NET Framework 2.0

·         .NET Framework 3.5 SP1

·         SQL Management Studio ile Express SQL Server 2008

·         PHP 5.x

·         Ek kitaplıklar ve programlar

 

Hangi ürünlerin kurulacağı hakkında daha fazla bilgi için aşağıdaki makalelere başvurun:

Windows Server 2003'ü WebsitePanel Kurulumu için hazırlayın

Windows Server 2008'i WebsitePanel Kurulumu için hazırlayın

Web PI'yi aşağıdaki konumdan yükleyebilirsiniz:

http://www.microsoft.com/web/downloads/platform.aspx

 

Not Windows Server 2003'te kurulum işlemi .NET Framework 2.0 kurmak için izninizi isteyebilir. Bu kuruluma izin verme seçeneğini seçin.
