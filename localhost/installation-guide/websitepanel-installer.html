WebsitePanel Installer

Translations:

  • tr-TR

WebsitePanel Installer (WSPI) is a program for installing and upgrading WebsitePanel components. WSPI has a small size and it does not contain WebsitePanel components installation files, but those files will be downloaded during WebsitePanel installation or upgrade.

In order to install WebsitePanel you have to install WSPI on the server. Go to www.websitepanel.net, then click on Downloads menu item on top menu.

Download and install WebsitePanel Installer:

After WSPI installation is completed click “WebsitePanel Installer” shortcut on server desktop to run WSPI, then click ‘View Available Components’:

Note: WebsitePanel Installer must be run as an adminstrator. If it doesn’t open after clicking on the shortcut, in Windows Explorer open WebsitePanel.Installer.exe Properties –> Compatibility and check “Run this program as an administrator” checkbox.
