WebsitePanel 1.1.1 Beta Released

What's new:

Added: SSL Management (contributed by OnionMan)
Added: Domain Aliases changes Domain aliases changed to allow using existing DNS domains to setup aliases and pointers (contributed by yoda-ict)
Added: “Announcements” posting mode for Mail Lists in hMail Server Service Provider;
Added: SMTP authentication for Mail Lists in hMail Server Provider;

Fixed: issue with setting up a correct type handler for WebsitePanel.IisModules.SecureFolders HTTP authentication module in IIS 7 (contributed by  yoda-ict);
Fixed: issue with Mail List members removed after clicking “Save List” button in hMail Server Service Provider;
Fixed: MailAlias is disabled in hMail Server after alias settings update in WSP
Fixed: WSP incorrectly sets Forwarding Address for hMail mail account
Fixed: Exchange 2010 module: search needs to eliminate contacts from its checks
Fixed: Unable to remove subscribers from the list/group for Merak Mail Server

To install WSP 1.1.1 Beta, open WSP Installer configuration file (c:\program files (x86)\SMB SAAS Systems\WebsitePanel Installer\WebsitePanel.Installer.exe.config) and change "configuration\installer\settings\Web.Service" value to: http://www.websitepanel.net/services/InstallerService-beta.asmx.

Please note that after Beta build is installed, there will be no possibility to roll it back to previous release. Microsoft provides paid support for WebsitePanel releases, but not for Betas.
