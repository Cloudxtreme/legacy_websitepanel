WebsitePanel Dil Paketleri Oluşturma ve Uygulama

Bu kılavuz, "WebsitePanel Yerelleştirme Araç Seti" aracını kullanarak WebsitePanel için nasıl yeni bir dil paketi oluşturulacağını ve uygulanacağını açıklamaktadır.

 

Gereken Yazılımlar

Microsoft .NET Framework 2.0 ya da üzeri.

Microsoft .NET Framework Version 2.0 Yeniden Dağıtılabilir Paket (x86)

Microsoft .NET Framework Version 2.0 Yeniden Dağıtılabilir Paket (x64)

 

Araç Setini Kurma

WebsitePanel Yerelleştirme Araç Setinin son sürümünü buradan yükleyin:

 

https://sourceforge.net/projects/websitepanel/files/Releases/WebsitePanel/1.0/Tools/WebsitePanelLocalizationToolkit10.msi/download

 

Kurulum programını çalıştırın ve WebsitePanel Yerelleştirme Araç Setini kurmak için tüm talimatları uygulayın.

 

WebsitePanel Portalın aynı sürümünü yerelleştirmek için WebsitePanel Yerelleştirme Araç Setinin özel sürümü kullanılmalıdır; örneğin, WebsitePanel Portal 1.0.1'i yerelleştirmek için WebsitePanel Localization Toolkit 1.0.1 kullanılmalıdır.

 

Windows Vista / Windows Server 2008'de Araç Setini çalıştırma

WSP Yerelleştirme Araç Setini Windows Vista veya Windows Server 2008 üzerinde çalıştırırsanız uygulamayı Yönetici hesabıyla çalıştırmanız gerekir (explorer'da sağ tıklatın ve "Yönetici Olarak Çalıştır" seçeneğini tıklatın).

 

Araç Setini Kullanma

Yeni Yerel Ayar Ekleme

İlk adım, yeni bir yerel ayar oluşturmaktır. Üst bölmede Ekle... düğmesini tıklatın. Açılan pencerede yeni yerel ayarı seçin ve Tamam düğmesini tıklatın.

 

Kaynak Dizelerini Yerelleştirme

Sonraki adım, tüm kaynak dizelerini yerelleştirmektir. Tüm dizeleri belirttiğiniz yerel ayara dönüştürmeniz gerekir.

 

Dil Paketi Oluşturma

Yerelleştirme sonrasında, bir dil paketi oluşturmanız gerekecektir. Tüm değişiklikleri kaydetmek için Kaydet düğmesini tıklatın ve yeni bir dil paketi oluşturmak için Dil Paketi Derle düğmesini tıklatın. Bu, dile özgü dosyalar oluşturacak ve bunları araç seti taban dizininin LanguagePacks alt dizinine sıkıştıracaktır.

 

Dil Paketi Uygulama

Dil paketi dosyalarını WSP Portalı ana dizinine açın ve WSP Portal'ın \App_Data\SupportedLocales.config dosyasını değiştirin:

 

<?xml version="1.0" encoding="utf-8" ?>

<SupportedLocales>

    <Locale name="English" key="en-US" fallback="" />

    <Locale name="German" key="de-DE" fallback="" />

</SupportedLocales>
