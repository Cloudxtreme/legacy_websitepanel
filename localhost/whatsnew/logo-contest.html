Logo Contest

Add a logo to the WebsitePanel project!!

The WebsitePanel team would like the WebsitePanel user community handpick the logo for their favorite product. We are calling all designers in the WebsitePanel community to contribute a logo for the project.

Submit your logo designs by 31th Aug. Follow ‘How To Submit your Logo Design’ below.

All WebsitePanel users can vote for their favorite logo. The logo with the most votes will be selected to represent WebsitePanel.

Cast your votes by 15st Sept. Follow ‘How To Vote For Your Favorite Logo’ below.

Get involved today!

 

How To Submit your Logo Design?

1. Submit logos on CodePlex Issue Tracker.

2. Create a new item and add an appropriate Title, for example "Logo: Contributor Name - Unique title for Logo".

3. Be sure to provide Work Item Details for correct categorization of the Logo. (CodePlex does not currently allow non-members to edit the Work Item Details. A request has been raised to add this in.)

Choose the below options:

Status: Proposed

Type: Task

Component: Choose Project Logo.

Custom: Enter a custom message if required.

4. Attach File. Be sure to attach your proposed logo so other users can view and vote for it. To ensure correct upload please stay within the limit of 4MB.

 

How To Vote For Your Favorite Logo?

1. Find the logo contributions at CodePlex Issue Tracker.

2. Choose the below options:(CodePlex does not currently allow non-members to edit the Work Item Details. A request has been raised to add this in.)

Status: Proposed

Type: Task

Component: Choose Project Logo.

3. Open tasks to find logo attachments.

4. Vote for your favorite logo
